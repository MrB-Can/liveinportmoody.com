- [ ] Cookie preferences mentioned → good for PIPEDA compliance

### Use of Information
- [ ] Responding to inquiries → primary use case ✓
- [ ] Routing buyer/seller/relocation/market leads → matches business purpose ✓
- [ ] SMS follow-up with consent → **verify this language:**

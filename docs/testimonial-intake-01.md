# Testimonial Intake — First Batch (up to 3)

**Purpose:** Collect and verify the first approved client testimonials for the `/raving-fans` page.  
**Rule:** Every field must be real, owner-approved, and sourced. Do not invent, paraphrase, or summarise without explicit written client consent.

---

## How This Works

1. Owner identifies 1–3 clients willing to appear publicly.
2. Owner collects written consent from each client (email or message is fine).
3. Owner fills in this doc with the exact approved text — no paraphrasing.
4. Development team adds approved entries to `/raving-fans` only after all items are confirmed.

The page currently accepts three source types: **Google reviews**, **RankMyAgent / RateMyAgent profile links**, and **client-approved stories**. Each testimonial below should specify which source type applies.
